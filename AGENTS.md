# Project Instructions

本项目是网联清算有限公司业务场景知识库生成工具。所有生产内容和提供的原始文件默认处于网联清算有限公司业务范畴内，生成内容中不需要反复说明这一全局背景。

本项目可能用于生成网联清算有限公司不同业务场景下的知识库。通用逻辑代码不得仅依据当前 `input` 内容硬编码或耦合某个单独业务场景的关键词、章节名、产品名、规范名或术语，例如当前样本文档中的具体业务场景词。业务关键词、分类语义、关联关系应优先由原文和大模型理解生成，程序只负责通用结构、格式、合法性、去重和质量约束。

关键词过滤只能处理跨场景的结构性噪声，例如固定章节标题、空值占位、输出模板字段等；不得在通用过滤逻辑中写入某个业务域、某份规范、某个接口或某个场景专属词。

后续修改完成后，只验证当前仓库代码和生成结果。除非用户明确要求，不要复制文件到其他项目，不要重建其他项目的向量库。
